## 🚀 PROCHAINES ÉTAPES (5 minutes)

### 1. Aller sur Vercel
👉 **https://vercel.com**

### 2. Créer le Projet
- Cliquez sur "Add New Project"
- Sélectionnez `DringDring`
- **Root Directory** : `frontend` ⚠️

### 3. Ajouter les 8 Variables
Voir `VERCEL_DEPLOY_CHECKLIST.md` pour la liste complète

### 4. Déployer
- Cliquez sur "Deploy"
- Attendez 2-5 minutes
- Notez l'URL Vercel

### 5. Mettre à Jour CORS
```powershell
heroku config:set CORS_ORIGIN=https://VOTRE-URL-VERCEL.vercel.app -a dringdring-backend
heroku restart -a dringdring-backend
```
