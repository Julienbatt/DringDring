# DringDring

Plateforme territoriale de livraison durable et de gouvernance logistique.

Ce depot contient :
- la documentation de vision (docs/)
- les roles et guards backend (docs/ROLES.md)
- le guide dev local (docs/DEV_LOCAL.md)
- le runbook dev (docs/RUNBOOK_DEV.md)
- le backend (FastAPI)
- le frontend (Next.js)

Le projet est reconstruit from scratch selon une approche orientee
collectivites, tarification gouvernee et multi-acteurs.
